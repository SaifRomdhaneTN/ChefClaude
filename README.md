This project was created following Section 3 of the Learn React Course at [Scrimba.com]()

It teaches the notion of React State, and it uses a HuggingFace AI API to generate a recipe from ingredients

You will have to add a .env file and fill it as follows: VITE_API_TOKEN="YOUR_API_TOKEN".
and change the YOUR_API_TOKEN with your own from Hugging Face
